# ProgettoBiblioteca

## Gruppo 3

- Pasquale Amore
- Pasquale Biele
- Leonardo Elefante
- Antonio Franco


Applicazione didattica per la gestione di una biblioteca (libri, utenti e prestiti), sviluppata come progetto universitario per esercitare concetti di basi di dati, programmazione ad oggetti e gestione delle transazioni.

## Funzionalità

- Gestione anagrafica dei libri (titolo, autore, anno, genere, disponibilità).
- Gestione degli utenti iscritti alla biblioteca.
- Registrazione, proroga e chiusura dei prestiti.
- Ricerca di libri per titolo, autore o altri criteri.
- Semplice interfaccia per operazioni di base della biblioteca (prestito, restituzione, consultazione catalogo).

## Struttura del progetto

- `src/` – Codice sorgente dell’applicazione.
- `docs/` – Documentazione aggiuntiva (Diagrammi delle classi, diagrammi delle sequenze e diagrammi dei package)
- `html/` – Documentazione automatizzata generata tramite Doxygen.
- `test/` – Test automatizzati tramite JUnit.
